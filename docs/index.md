# itd-sdk

SDK для работы с [итд.com](https://итд.com)

## Про ИИ
98% всего кода и документации было написано человеком.
ИИ только написал парсинг `html` (`itd.utils`) а также ВОЗМОЖНО некоторые примеры в `examples/` (пул-реквесты с примерами писал не я). Также пару раз использовался для код-ревью (проверить код).
