parse_html 

parse_html(text: str) -> tuple[str, list[Span]]

Парсит HTML-текст, извлекает чистый текст и spans с форматированием.

Поддерживаемые теги: - <b>, <i>, <s>, <u>, <code>, <spoiler>, <q> - <a href="url">text</a> или <а>url</а> (ссылки)

parse_md 

parse_md(s: str) -> tuple[str, list[Span]]

Парсит markdown, извлекает чистый текст и spans с форматированием.

жирный, курсив, ~~зачёркнутый~~, подчёркнутый, моноширный, ||спойлер||, >цитата>

Теги могут пересекаться, например: __1111234__32342

Теги могут быть вложенными: 12311323

[текст ссылки](url)

Внутри ссылки остальные теги не парсятся

Если url пустой, на его место будет поставлен текст ссылки, то есть:

[текст ссылки]() = [текст ссылки](текст ссылки)

используйте \ для экранирования символов

Parameters:

s str

 required

Returns:

чистая строка
list[Span]

list[Span]: список спанов
